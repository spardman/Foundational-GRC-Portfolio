# Information Security Policy (ISP)

## 1. Purpose & Scope
This policy defines the high-level security requirements and governance framework for protecting [Company Name]'s information assets, systems, and personnel.

## 2. Framework Alignment
This policy is aligned with **NIST Cybersecurity Framework (CSF)** and **ISO 27001** standards to ensure an industry-standard security posture.

## 3. Core Policy Statements
*   **Access Control:** Access to systems is granted based on the principle of **Least Privilege**.
*   **Data Integrity:** All sensitive data must be encrypted at rest and in transit.
*   **Asset Management:** Only authorized devices are permitted to connect to the corporate network.

## 4. Roles & Responsibilities
*   **Security Advisor:** Responsible for incident response and policy maintenance.
*   **Management:** Accountable for resource allocation and risk acceptance.
*   **Users:** Responsible for adhering to the **Acceptable Use Policy (AUP)**.
